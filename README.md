## Getting Started
### Prerequisites
* Streaming software like [OBS](https://obsproject.com/)
* [Docker](https://www.docker.com/)

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/VishwanathDalawai/Go-live.git
   ```
3. Docker build
   ```sh
   docker compose build
   ```
4. Docker compose
   ```js
   docker compose up
   ```
